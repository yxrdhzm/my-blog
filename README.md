# 月下人独 - 个人博客网站

## 项目概述

这是一个简洁的个人静态博客网站，主要用于展示壁纸和视频资源，并提供简单的内容访问控制。网站采用单页应用设计，使用纯HTML、CSS和JavaScript实现，无需后端支持即可部署运行。

### 主要功能

- 响应式导航系统，支持在不同页面之间无缝切换
- 壁纸和视频资源的网格布局展示
- 简单的密码保护机制，控制对敏感内容的访问
- 全屏背景图片效果增强视觉体验
- 自适应不同设备屏幕尺寸的响应式设计
- GitHub Pages自动化部署配置

### 技术栈

- **前端技术**：HTML5, CSS3, JavaScript ES6
- **部署平台**：GitHub Pages
- **CI/CD工具**：GitHub Actions

### 页面结构

网站采用单页应用架构，通过JavaScript控制不同内容区块的显示和隐藏：

1. **首页 (`#home`)** - 网站入口页面，展示博客标题和简短介绍
2. **壁纸分享引导页 (`#wallpapers`)** - 提供密码验证入口的引导页面
3. **壁纸画廊 (`#gallery`)** - 受密码保护的图片和视频展示页面
4. **关于我 (`#about`)** - 博主信息介绍页面

### 核心功能模块

#### 1. 页面导航系统

- 使用`navigate()`函数控制页面切换
- 通过`active`类管理当前显示的页面和导航菜单项
- 支持点击导航菜单或页面内按钮进行跳转

#### 2. 密码验证机制

- 弹窗式密码输入界面
- 客户端密码验证（仅用于演示目的，生产环境建议使用服务端验证）
- 密码正确后才显示受保护的壁纸画廊内容

#### 3. 媒体资源展示

- 响应式网格布局，自动适应不同屏幕尺寸
- 图片和视频混合展示，统一的视觉样式
- 视频互斥播放功能，确保同一时间只有一个视频播放

## 开发与部署指南

### 本地开发

1. **克隆项目**

   ```bash
   git clone https://github.com/your-username/my-blog.git
   cd my-blog
   ```

2. **本地预览**

   直接在浏览器中打开`index.html`文件即可查看效果，无需额外的构建或服务器。

### 部署说明

项目已配置GitHub Actions自动部署：

1. **部署触发条件**：推送到`main`分支时自动启动部署
2. **部署目标**：GitHub Pages
3. **部署配置**：详见`.github/workflows/deploy.yml`文件

### 自定义与扩展

#### 修改内容

- 编辑`index.html`中的文本内容可以更新博客标题和介绍
- 在`images`目录添加/替换图片和视频资源
- 修改JavaScript中的密码（默认密码：123456）

#### 样式定制

- 所有CSS样式都在`index.html`的`<style>`标签内
- 可以修改颜色、字体、布局等样式参数
- 响应式断点设置在768px和480px

## 代码说明

### 关键JavaScript函数

- **`navigate(pageId)`** - 页面切换函数
- **`showPasswordModal()`** - 显示密码验证弹窗
- **`closeModal()`** - 关闭密码验证弹窗
- **`checkPassword()`** - 验证密码并控制访问权限

### CSS主要部分

- 全局样式重置与基础设置
- 固定顶部导航栏样式
- 页面容器和内容布局
- 响应式设计适配
- 模态弹窗样式

## 安全注意事项

⚠️ **重要安全提示**：当前实现的密码保护仅在客户端进行，主要用于演示目的。在生产环境中，强烈建议：

- 使用服务端身份验证替代客户端密码验证
- 移除代码中的硬编码密码
- 考虑使用HTTPS确保传输安全

## 性能优化建议

- 压缩图片资源以提高加载速度
- 考虑实现图片懒加载
- 为图片提供WebP格式支持
- 添加适当的缓存控制策略
